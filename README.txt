Getting Started:
	Prerequisites:
	1. python 3+
	2. install requests HTTP library 
	   command: python -m pip install requests
	3. install netaddr library
	   command: python -m pip install netaddr
	4. Any Operating System (MacOS, Linux, Windows)

	Project Description:

		Contents:
        1. Main.py - Entrypoint for the project
        2. BottleRocket.py - Contains the core functionalities
        3. BottleRocketConstants.py - Class containing the constant values
        4. BottleRocketHelpers.py - Helper class to support BottleRocket.py
        5. BottleRocketTest.py - Unit test cases covering the scenarios
        6. README.txt

Note:
**Implemented the extra credit functionality in BottleRocket.py**

How To Run?
    1. From the project Directory location, run "python Main.py <region>"
    2. <region> - can contain only numbers, alphabets, dashes
    3. Main.py can accept only one argument (as mentioned in the requirements section)
